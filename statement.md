# Problem Statement
Diabetes is a major health concern worldwide. Early detection can save lives. 
This project predicts the likelihood of diabetes in individuals based on health data.

# Scope of the Project
- Analyze relevant health indicators
- Train a machine learning model to predict diabetes
- Provide a user-friendly GUI for non-technical users to input data and get predictions

# Target Users
- Individuals interested in personal health monitoring
- Healthcare professionals for preliminary screening

# High-Level Features
- Input fields for key health indicators
- Prediction with confidence score
- Clear, easy-to-use GUI
- Model trained and saved for reuse
