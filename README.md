# Diabetes Prediction GUI
## Overview
This project predicts whether a person is likely to have diabetes based on several health indicators. 
It uses a trained machine learning model and provides a user-friendly Tkinter GUI for interaction.

- Developed by: **Vaishnavi Mishra**
- Tools used: Python, scikit-learn, NumPy, Pandas, Tkinter, Joblib

## Features
- Input health data (Glucose, BMI, Blood Pressure, etc.)
- Predict diabetes with a confidence percentage
- Clear button to reset inputs
- Simple, user-friendly GUI

## How to Run
1. Clone the repository.
2. Ensure `diabetes_model.pkl` is in the same folder as `diabetes_gui.py`.
3. Install required packages: pip install -r requirements.txt
4. Run the GUI: python diabetes_gui.py
